# Projet Système Réparti

Ce projet consiste à déployer une application web répartie basée sur une architecture
microservices en utilisant Docker, Kubernetes, Ansible et Jenkins.

## Objectifs
- Concevoir une architecture microservices
- Conteneuriser une application web
- Déployer sur Kubernetes
- Automatiser l’infrastructure avec Ansible
- Mettre en place un pipeline CI/CD avec Jenkins

## Architecture
L’application est composée de trois services :
- Frontend : React avec TypeScript
- Backend : Django REST Framework
- Base de données : PostgreSQL

La communication entre les services se fait via des API REST.
Le déploiement est automatisé et orchestré via Kubernetes.

## Technologies utilisées
- React + TypeScript
- Django + Django REST Framework
- PostgreSQL
- Docker & Docker Compose
- Kubernetes (Minikube)
- Ansible
- Jenkins
